# Netflix Streaming Data Analysis using SQL and Power BI
## Project Objective
The objective of this project is to analyze Netflix-style streaming data to understand user behavior, content performance, and subscription trends using SQL and Power BI.
This project aims to transform raw data into actionable insights by performing SQL-based analysis and creating interactive dashboards that help identify viewing patterns, user retention, and content preferences.

# Dataset Used 
<a href="https://drive.google.com/drive/folders/1TpIfHTMhyyk3096GNSSh2at23hTfRVHP?usp=drive_link">Netflix Dataset</a>

# Questions (KPIs using SQL)
- Total Users, Subscriptions & Revenue
- Average Order Value (AOV)
- New vs Existing Customers
- Repeat User %
- MoM Revenue & Subscription Trends
- Purchase Patterns (Day & Time)
- Avg Watch Duration
- Most Consumed Content
- Device/Platform Usage
- Indian vs Foreign Content %
- Live vs Relegated Content %
- Content Performance after Release
