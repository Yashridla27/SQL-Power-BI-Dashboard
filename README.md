# Netflix Streaming Data Analysis using SQL and Power BI
## Project Objective
The objective of this project is to analyze Netflix-style streaming data to understand user behavior, content performance, and subscription trends using SQL and Power BI.
This project aims to transform raw data into actionable insights by performing SQL-based analysis and creating interactive dashboards that help identify viewing patterns, user retention, and content preferences.
